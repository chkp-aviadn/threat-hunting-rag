# Architecture Diagrams

This directory contains system architecture documentation:

## Files
- `architecture.mmd` - Mermaid diagram showing system flow and layers

## Viewing Diagrams
```bash
# Install mermaid CLI (optional)
npm install -g @mermaid-js/mermaid-cli

# Generate PNG from mermaid
mmdc -i architecture.mmd -o architecture.png
```

Or view directly in GitHub, VS Code with Mermaid extension, or online at https://mermaid.live
